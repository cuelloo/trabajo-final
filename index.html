1. ¿Qué es CSS?

CSS son las siglas de “Cascading Style Sheets” (hojas de estilo en cascada). CSS es un lenguaje para la composición y estructuración de páginas web (HTML o XML). Este lenguaje contiene elementos de codificación y se compone de estas “Cascading Style Sheets” que también se llaman archivos CSS

2. ¿Cuál es la función principal de CSS?

el código en CSS se encarga de darles formato y presentarlos visualmente a través de un navegador

3. ¿Cómo se puede incluir CSS en un documento HTML?

// link rel="stylesheet" href="style.css"

4. ¿Qué son los selectores en CSS?

Un selector CSS es la primera parte de una regla CSS. Es un patrón de elementos y otros términos que indican al navegador qué elementos HTML se seleccionan para aplicarles una regla que incluye los valores de las propiedades CSS.

5.¿Qué son las propiedades y valores en CSS?

Las propiedades CSS no son más que aquellas características a las que vamos a asignar valores que definirán el estilo concreto de nuestros elementos html.

6. ¿Cómo se aplica el modelo de caja en CSS?

Cuando hablamos del modelo de cajas en CSS, estamos haciendo referencia a un sistema que tiene el navegador de interpretar las diferentes partes de lo que solemos denominar «caja»: un elemento HTML con unas ciertas dimensiones.La representación básica del modelo de cajas se basa en varios conceptos importantes, como veremos a continuación: El borde (border), en negro, es el límite que separa el interior del exterior del elemento. El márgen (margin), en naranja, es la parte exterior del elemento, por fuera del borde. El relleno (padding), en verde, es la parte interior del elemento, entre el contenido y el borde. El contenido, en azul, es la parte interior del elemento, excluyendo el relleno.

7. ¿Qué es el "cascading" en CSS?

Se le denomina estilos en cascada porque se lee, procesa y aplica el código desde arriba hacia abajo

8. ¿Qué son las media queries?

Las media queries son una técnica en CSS que permite aplicar diferentes estilos a diferentes dispositivos basados en sus características, como el tamaño de la pantalla. Son esenciales para el diseño web responsive

9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?

El CSS en línea: Esta técnica consiste en aplicar estilos directamente en la etiqueta HTML utilizando el atributo ‘style’. Aunque es útil para cambios rápidos, puede dificultar el mantenimiento y la escalabilidad del código. El CSS interno: Mediante la inclusión de reglas CSS dentro de la etiqueta El CSS externo: Al crear archivos .css separados que contienen todas las reglas de estilo, se promueve la reutilización del código y se facilita su gestión en proyectos más grandes. El CSS en cascada: Este principio fundamental de CSS establece cómo se resuelven los conflictos cuando múltiples reglas afectan a un mismo elemento, lo que permite controlar la prioridad y especificidad de los estilos. El CSS responsivo: Con el auge de los dispositivos móviles, el diseño responsivo se ha vuelto indispensable. Mediante media queries y unidades relativas como porcentajes o ems, es posible adaptar la apariencia de un sitio web a diferentes tamaños de pantalla.
